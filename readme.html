This is the readme for the model associated with the paper:

 Ferguson KA, Huh CY, Amilhon B, Williams S, Skinner FK (2013)
 Experimentally constrained CA1 fast-firing parvalbumin-positive
 interneuron network models exhibit sharp transitions into coherent
 high frequency rhythms. Front Comput Neurosci 7:144

This model was contributed by F. Skinner.
